# Jenkins Pipelines
Pipelines for common use

## gitlab-branch-lock-unlock
Lock and unlock GitLab branches with the help of git API's
  ### Pre-requisites:-
    1. Gitlab API Token
    2. Jenkins Credentials
    3. identify the GitLab project id's

## Gitlab API Token
https://docs.gitlab.com/ee/user/profile/personal_access_tokens.html

## Jenkins Credentials
https://www.jenkins.io/doc/book/using/using-credentials/
